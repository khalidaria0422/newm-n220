# Simple Calculations Assignment

Web 4: [Assignment 2](https://in-info-web4.luddy.indianapolis.iu.edu/~karia/newm-n220/assignment2)

## Instructions

For this homework assignment, you will implement JavaScript to solve various arithmetic problems. The images below illustrate the final results.

To complete this homework, complete the following:

1. Create a folder called homework-two.
2. Download the beginning [html file](https://iu.instructure.com/courses/2402404/files/206751134?wrap=1) and place it in your folder.
3. Create a separate javascript file and link it to the html page.
4. Complete the problems on the html file.
5. Pizza Problem Hint: For the pizza problem, you will get a decimal for the number of students fed. Obviously, you cannot have less than a whole person, so you will need to round this down. To do this, use the code below and REPLACE students with your decimal result:
   `Math.floor(student);`
6. Upload your project to the web 4 server.
7. Submit your URL for the assignment submission.
